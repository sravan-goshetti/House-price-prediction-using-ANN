# House Price Prediction using Artificial Neural Network

## Project Overview
This project is about prediction of House price using Artificial Nural Networks. 
 
This repository consists of contnts of data-exploration, visuvalisation, building of models, fine tuning models

* **Visualization:** `matplotlib`, `seaborn`
* **Machine Learning:** `scikit-learn`, `statsmodels`, `imbalanced-learn`,`tensor flow` , `keras`

## Methodology

 The project follows a standard Machine Learning pipeline:
* Exploratory Data Analysis (EDA): Visualizing distributions and correlations.Making insights from the graphs.

* Preprocessing: Scaling numerical features and encoding categorical variables.

* Model Implementation: Implementing ANN, Visuvalising accuracy graphs, fine tuining model.
